Sobre o Projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026 com o tema *A Importância do Agronegócio*. O site apresenta informações sobre o agronegócio brasileiro, destacando sua história, sua importância para a economia, o uso da tecnologia no campo, a sustentabilidade e os desafios enfrentados pelo setor.

Objetivo

O objetivo deste projeto é conscientizar os visitantes sobre a importância do agronegócio para o desenvolvimento do Brasil, mostrando sua contribuição para a produção de alimentos, geração de empregos, crescimento da economia e preservação do meio ambiente por meio de práticas sustentáveis.


Justificativa

O agronegócio é um dos principais setores da economia brasileira e está presente no dia a dia de toda a população. Compreender sua importância permite valorizar o trabalho realizado no campo e reconhecer como a tecnologia e a sustentabilidade contribuem para uma produção agrícola mais eficiente e responsável.

Funcionalidades

O site possui as seguintes funcionalidades:

- Página responsiva para computadores, tablets e celulares;
- Botão para ativar e desativar o modo escuro;
- Botão "Quero Saber Mais";
- Navegação simples e intuitiva;
- Efeitos de hover nos botões;
- Transições suaves utilizando CSS;
- Layout organizado utilizando Flexbox;
- Conteúdo dividido em seções.

Implementação

Durante o desenvolvimento foram utilizados recursos como:

- HTML semântico para organizar o conteúdo;

- Classes e IDs para estilização específica;
- JavaScript para manipulação do DOM;
- Eventos de clique para interação com o usuário.
